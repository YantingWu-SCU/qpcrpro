# 🧬 qPCR-Pro: RT-qPCR 实验数据分析工具

专为生物材料/生物医学博士设计的轻量级网页工具，用于快速处理 qPCR 原始数据。

## ✨ 主要功能

1.  **自动计算**：只需输入原始 Ct 值，自动进行内参归一化和对照组校正。
2.  **GraphPad Prism 友好**：直接生成“列（Column）”格式数据，**一键复制**后可直接粘贴进 Prism 作图（带散点的柱状图）。
3.  **内置统计分析**：
    * **2组数据**：自动进行 T 检验 (T-test)。
    * **3组及以上**：自动进行单因素方差分析 (One-way ANOVA) 并提供 **两两比较矩阵 (Pairwise Matrix)**，P 值显著性 (*, **, ***) 一目了然。
4.  **隐私安全**：所有计算均在浏览器本地完成（Pure JavaScript），数据不会上传到任何服务器。

## 🚀 如何使用

### 1. 准备数据
从你的 qPCR 仪器导出 Excel 文件。你需要每个样品、每个基因的原始 Ct 值（通常每组 3 个复孔）。

### 2. 打开工具
访问工具链接。

### 3. 生成表格
* 输入 **实验组数量** (例如: Control, Hydrogel, Hydrogel+BMP2 共3组)。
* 输入 **基因数量** (例如: GAPDH, Runx2, OCN 共3个)。
* 点击 **“生成表格”**。

### 4. 录入数据
* **修改名称**：在表头修改基因名，在左侧修改组名。
* **粘贴数据**：直接从 Excel 复制 3 个复孔的 Ct 值，粘贴到对应的框里。
    * *格式支持*：Excel 默认的换行格式、逗号分隔、空格分隔均可。
    * *注意*：**内参基因（如 GAPDH）必须每一行都填入数据**，否则无法计算！

### 5. 计算
* 在下方下拉菜单选择哪一列是 **内参基因**。
* 选择哪一组是 **对照组 (Calibrator)**。
* 点击绿色按钮 **“开始计算”**。

### 6. 结果解读与作图
* **左侧图表**：快速预览趋势。
* **右侧文本框**：点击“一键复制”，去 GraphPad Prism 新建一个 `Column` 表，粘贴即可。
* **下方矩阵**：查看 P 值。例如 `Hydrogel` vs `Control` 的 P < 0.01 (**)，说明有显著差异。
